# 📒 Agenda de Contatos em Python

Este é um projeto simples de linha de comando feito em Python que permite adicionar, listar e remover contatos. Os dados são salvos em um arquivo `.txt`, permitindo persistência das informações.

## 🚀 Funcionalidades

- Adicionar um novo contato (nome, endereço e telefone)
- Listar todos os contatos salvos
- Remover todos os contatos
- Interface simples no terminal
